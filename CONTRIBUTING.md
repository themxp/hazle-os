# Contributing

Want to help? Here's how.

## Getting Started

1. Fork the repo
2. Make changes
3. Test in QEMU
4. Submit pull request

## Code Guidelines

- Keep it simple and readable
- Use 4 spaces for indentation
- Use `snake_case` naming
- Comment only complex parts
- Write in English

## Commit Format

```
type: description

Optional longer explanation
```

Types: `feat`, `fix`, `docs`, `refactor`

Example:
```
feat: add RTC driver

Reads time from CMOS real-time clock
```

## Before Submitting

- Code compiles without errors
- Tested in QEMU
- Existing features still work
- Updated docs if needed

## Questions?

Open an issue.
